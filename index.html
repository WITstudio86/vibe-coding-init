一键初始化

自动创建 Git 仓库、.vibe/ 目录、角色分配文件、跨会话消息总线，5 秒完成全部配置。

🤖

自动创建会话

通过 ZCode CLI 自动创建 5 个角色会话，自动注册到 GUI 任务列表，标题清晰展示角色名。

📨

跨会话通信

HANDOFF.md 消息总线 + ReadSessionContext 上下文拉取，角色间无缝协作。

🔀

双入口工作流

新功能走需求分析 → 设计 → 编码 → 验证；Bug 修复走验证 → 分析 → 编码 → 验证。

⚡

Superpowers 集成

每个角色预设最佳 Superpower 武器库：brainstorming、TDD、systematic-debugging、verification 等。

📋

标准产出文件

SPEC.md · DESIGN.md · BUGFIX.md — 每个角色有明确的交付物和验收标准。

快速开始

三步开始协作

1. 安装 Skill

# 克隆到用户 skills 目录
git clone https://github.com/WITstudio86/vibe-coding-init.git ~/.agents/skills/vibe-coding-init/

2. 触发初始化

# 在 ZCode 中打开项目目录，输入：
初始化 vibe coding 项目

3. 自动创建会话（需 Node ≥ 22）

# Skill 自动执行以下流程：
git init
mkdir -p .vibe
# 创建 ROLES.md + HANDOFF.md
# 创建 5 个 ZCode 会话（需求分析 / Bug修复 / 技术设计 / 代码编写 / 功能验证）
# 注册到 GUI 任务列表
