⚡

秒级创建

直接写 SQLite 数据库，4 个角色会话 < 0.2 秒完成，无需模型 API 调用。

🤖

/trigger 自动触发

自定义 ZCode 命令，一行 /trigger sess_xxx "任务" 唤醒下游会话，无需手动拼 Bash。

📨

消息总线

HANDOFF.md 记录所有跨会话通信，ReadSessionContext 拉取完整上下文。

📋

标准产出文件

SPEC.md · DESIGN.md · REVIEW.md · BUGFIX.md —每个角色有明确交付物。

🛡️

角色边界隔离

每个角色有明确的 ✅/❌ 边界——调试员一行代码都不改，审查员只验证不修改。

🖥️

GUI 友好

会话标题显示角色名，不被首条消息覆盖，重启后即可看到整洁的会话列表。

快速开始

三步开始协作

1. 安装

# 安装 skill
cp SKILL.md ~/.agents/skills/vibe-coding-init/

# 安装 /trigger 命令
mkdir -p ~/.zcode/commands
cp scripts/trigger.md ~/.zcode/commands/

2. 初始化

# 在 ZCode 项目目录中输入：
初始化 vibe coding 项目

3. 自动协作

# 每个角色完成后，用 /trigger 唤醒下游
/trigger sess_xxx "请读取 DESIGN.md，TDD编码实现"
